Guide d'installation de l'environnement

Cette partie concerne maintenant l'installation d'outils que vous utiliserez pour travailler sur votre machine et sur Pepper. Premièrement, commencez par télécharger les fichiers suivants. Ces fichiers vous amèneront sur la page web de softbankrobotics Vous pourrez vous y connecter avec ce mail robotique@consertoteam.pro Référez-vous à Emmanuel pour obtenir le mdp

SDK Python SDK C++ Cross Toolchain Choregraphe 2.5.5

Lancer ensuite le script d'installation nommé install_env.sh lorsque le téléchargment des SDKs est terminé 

                    sudo ./install_env.sh

À l'ouverture de votre deuxième terminal, le script createEnv.sh est éxécuté. Tapez successivement 1 (Unix Makefiles), 2 (QtCreator), Y (pour confirmer)
