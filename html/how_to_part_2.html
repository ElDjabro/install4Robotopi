Guide d'installation de l'environnement

Cette partie concerne maintenant l'installation d'outils que vous utiliserez pour travailler sur votre machine et sur Pepper. Elle est donc essentielle au fonctionnement des applications. Premièrement, commencez par télécharger les fichiers en cliquant sur les liens suivants: (Note : Vous aurez besoin de cet identifiant robotique@consertotech.pro. Pour le mot de passe référez-vous à Emmanuel)

SDK Python SDK C++ Cross Toolchain Choregraphe 2.5.5

Lorsque le téléchargment des fichiers est terminé, tapez la commande suivante : 

                    sudo ./install_env.sh

À l'ouverture de votre deuxième terminal, le script createEnv.sh est éxécuté. Tapez successivement 1 (Unix Makefiles), 2 (QtCreator), Y (pour confirmer)
