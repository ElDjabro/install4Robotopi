Guide d'installation

First things first

Avant tout vérifiez que vous étes connectés à un réseau Wi-Fi fiable, puis ouvrez le Terminal

Creer un dossier caché ".conserto" (mkdir ~/.conserto) et tapez la commande suivante : 

                    
                        sudo apt-get update && sudo apt-get upgrade

Il vous faudra par la suite installer git, un gestionnaire de version très utile pour travailler en groupe. Mais je vous en parlerai en détail un peu plus bas ;) Pour l'instant tapez la commande : 

                    
                        sudo apt-get install git

Terminator : votre ami !

Fini les fonds d'écrans en noir et blanc et l'écriture illisible.Terminator est un terminal virtuel qui a la particularité de permettre de scinder en plusieurs parties la fenêtre selon vos envies et ainsi organiser plus simplement vos différentes fenêtres. Comme vous allez être amené à vous logger sur le robot, lancer l'application sur ce dernier et voir les logs d'erreurs, je vous conseille fortement d'installer ce terminal.

Installation de Terminator

                            sudo apt-get install terminator

Les commandes utiles

  • Ctrl-Shift-E : Scinder la fenêtre verticalement.
  • Ctrl-Shift-O : Scinder la fenêtre horizontalement.
  • Ctrl-Shift-N ou Ctrl-Tab : Déplacer le curseur dans la fenêtre suivante.
  • Ctrl-Shift-P ou Ctrl-Shift-Tab : Déplacer le curseur dans la fenêtre précédente.
  • Ctrl-Shift-W : Fermer la fenêtre courante.
  • Ctrl-Shift-X : Agrandir la fenêtre courante.
  • Ctrl-Shift-Flèche : Agrandir ou rétrécir la fenêtre courante.
  • Ctrl-Tab : Changer de "split" de fenêtre.

Configuration Terminator

Pour configurer Terminator, il faut éditer le fichier ~/.config/terminator/config (s'il n'existe pas il suffit de le créer)

Vim l'editeur de texte du super codeur

L'éditeur de texte est un outil de base sous Linux. Il sert notamment à modifier les fichiers de configuration du système. Les deux éditeurs de texte les plus connus et les plus utilisés sont Vim et Emacs. Vim signifie Vi IMproved : il s'agit d'une version améliorée du classique vi. Il est très complet, peu gourmand en ressources, et fait très bien la coloration syntaxique.

Installation de Vim

                            sudo apt-get install vim

Utilisation de Vim

Je vous laisse regarder sur le site d'openclassroom pour l'utilsation de Vim Utilisez Vim

Configuration Vim

Vous pouvez configurer votre éditeur préféré en lançant des commandes et certaines options méritent d'être activés. Cependant le fonctionnement des options sur vim peuvent être oublié dès que l'on quitte le logiciel. C'est pourquoi il faut créer un fichier de configuration .vimrc dans votre répertoire personnel...Mais on a pensé à tout ! :) Plus besoin de faire de réglages Aller sans plus attendre dans le dossier "config" et taper ces commandes:

Z Shell

Présentation Zsh

Très souvent lorsque l'on parle de programmation sur un environnement UNIX on est souvent amené à parler de shells, terminaux et de ligne de commandes. Le shell le plus répendu et par défaut sur linux reste Bash mais il existe d'autres variantes aussi bien utilisés comme Zsh ou Z Shell.
Les avantages du Z Shell sont nombreuses mais je ne vais pas m'aventurer là dedans. Neanmoins on peut citer les principaux:

  • Autocomplétionavancée
  • Correction orthographique des commandes saisies
  • Support avancé de Git
  • Historique partagé entre les différents Shell ouverts.

Installation Zsh et le mettre par défaut

  1. Pour installer Zsh, saisissez les commandes suivantes : 

                                        sudo apt-get install zsh

  2. Vérifiez l'installation en saisissant : 

                                        echo $(which zsh)

    vous devriez voir afficher sur le terminal /usr/local/bin/zsh

  3. En faire son shell par défaut: 

                                        sudo chsh -s $(which zsh)

Installation du framework Oh my Zsh

Vous pouvez à présent personnaliser votre shell avec ce framework, car il contient une part importante de plugins que vous pouvez installer. 

                            
                                sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

Personnalisation de votre shell

Dirigez-vous vers le dossier config/perso et modifiez les paramètres commentés dans le fichier gitconfig et zshrc. Vous y trouverez une liste non exhaustive de paramètres réglables. Lorsque que vous êtes sûr de vos changements tapez les commandes suivantes 

                            
                                chmod a+x install.sh
                                sudo ./install.sh

Surveiller l'activité système de l'ordinateur avec Indicator-Multiload

Permet de voir en direct la charge des processeurs, l'occupation en mémoire vive, l'activité de la connexion Internet et quelques autres paramètres, le tout sous forme de petits graphs qui se logent dans la barre du haut du bureau. Déjà dans les dépôts officiels d'Ubuntu, donc installation en une simple ligne de commande depuis le Terminal. 

                    sudo apt-get install indicator-multiload
